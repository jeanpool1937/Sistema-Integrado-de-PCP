# GEMINI.md - Memoria del Proyecto y Reglas de Vida

## 1. Regla Principal de Comunicación
- **Idioma Obligatorio**: Todas las respuestas, planes, comentarios y documentación deben estar estrictamente en **ESPAÑOL**.
- **Excepción**: Los nombres técnicos de variables, funciones y clases en el código se mantendrán en INGLÉS para preservar estándares internacionales, pero sus comentarios explicativos serán en español.

## 2. Propósito y Filosofía
- **Claridad**: Explicar conceptos técnicos de forma sencilla.
- **Seguridad**: Confirmar antes de ejecutar acciones destructivas.

## 3. Estándares Técnicos
- **Frontend**: React Functional Components, Hooks.
- **Backend**: Supabase (PostgreSQL), Python scripts para ETL.
